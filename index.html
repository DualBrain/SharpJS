Create cross-platform HTML5/JS apps with C#/.NET

About SharpJS

SharpJS is a framework that compiles C# code into HTML and JavaScript to run anywhere.

Its .NET runtime engine is powered by JSIL. While JSIL was written to easily port XNA games to run on the Web, SharpJS adds a set of APIs and classes for building GUI applications, with C# classes that are later mapped to native HTML5 elements. The HTML5 applications also utilize modern CSS frameworks such as Foundation 6 and Bootstrap.

SharpJS's simple structure makes it relatively straightforward to port existing Silverlight and Windows Forms applicaations directly to HTML5.

SharpJS is completely open-source! The source is available on GitHub. SharpJS is maintained by 0xFireball from ExaPhaser Industries.

Download SharpJS

SharpJS can be installed and used in numerous ways. You can install the NuGet package and build an application based on the templates here, install the Visual Studio extension, or build the source yourself and reference it in your application.
